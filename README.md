# BuildApps.ca — Jekyll Theme (Light “Mobile Systems”)

A lightweight Jekyll site/theme based on the BuildApps.ca CodePen design you approved.

Includes:
- **Background repeat fix** (`background-attachment: fixed`, no tiled overlays)
- Optional subtle texture (`body::before`) that does **not** tile
- Hub-first homepage sections: Tracks, Platforms, Guides, Shipping, About
- Small client-side search + quick filters (demo)

## Run locally
```bash
bundle install
bundle exec jekyll serve
```
Open: http://localhost:4000

## Customize
- Styles: `assets/css/style.css`
- JS: `assets/js/main.js`
- Navigation: `_data/nav.yml`
- Homepage: `index.md`
- Platform hubs: `_platforms/`
- Guides: `_guides/`
- Shipping references: `_shipping/`
